# Painel Copiar - Dr. Ortopedia 2.0

Painel web simples para embutir no Notion com `/embed`.

Repositorio: `https://github.com/rmdmmed-ops/botao-copiar-notion`

URL esperada no GitHub Pages:

`https://rmdmmed-ops.github.io/botao-copiar-notion/`

## Arquivos

- `index.html`: estrutura do painel e texto clinico.
- `styles.css`: visual do cartao e do botao.
- `script.js`: funcao de copiar para a area de transferencia.

## Como publicar no GitHub Pages

1. Envie estes arquivos para a raiz do repositorio no GitHub.
2. No GitHub, abra `Settings`.
3. Entre em `Pages`.
4. Em `Build and deployment`, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salve.

Depois de alguns minutos, o GitHub deve gerar esta URL:

`https://rmdmmed-ops.github.io/botao-copiar-notion/`

## Como usar no Notion

1. Copie a URL publicada no GitHub Pages.
2. No Notion, digite `/embed`.
3. Cole a URL.
4. Ajuste a altura do bloco para deixar o painel confortavel.
